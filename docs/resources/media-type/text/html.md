# `text/html`

## File Extensions

`html`, `htm`

## Magic Numbers

1. HTML Preamble prefix in upper caase
   | | 0 | 1 | 2 | 3 |
   | ---- | ---- | ---- | ---- | ---- |
   | Byte | 0x3c | 0x21 | 0x44 | 0x4f |
   | Char | < | ! | D | O |

2. HTML Preamble prefix in lower caase
   | | 0 | 1 | 2 | 3 |
   | ---- | ---- | ---- | ---- | ---- |
   | Byte | 0x3c | 0x21 | 0x64 | 0x6f |
   | Char | < | ! | d | o |
